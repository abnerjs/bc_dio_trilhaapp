## TrilhaApp
Um projeto feito em Flutter para o bootcamp da DIO.

## Como executar
- Clone o repositório
- Instale as dependências com `flutter pub get`
- Inicie o app com `flutter run`
- O email e senha para login são `admin` e `admin`
